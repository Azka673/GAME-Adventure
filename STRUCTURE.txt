CHRONO HUNTER — STRUKTUR FOLDER & TEKNOLOGI
============================================

TEKNOLOGI (bukan Java / bukan Minecraft mod)
--------------------------------------------
- HTML5      → index.html (struktur UI, canvas game, overlay kuis, menu)
- CSS3       → css/*.css   (tampilan HD, HUD, menu, loading, ending)
- JavaScript → js/*.js     (logika permainan, rangkaian adegan, state)

Game ini berjalan di browser (desktop & mobile). Bahasa utama: JavaScript.
Java (JDK) tidak dipakai kecuali Anda nanti mem-port ke Android native.

URUTAN PEMUATAN SCRIPT (penting)
---------------------------------
index.html memuat minimal dalam urutan ini:
  1. utils.js      — Typewriter harus tersedia sebelum loading/quiz/dimensi
  2. loading.js
  3. dialog.js, library.js, quiz.js, …
  4. game.js
  5. main.js       — memanggil LoadingScreen.show di akhir

Alur pengalaman pemain (skrip)
-----------------------------
main.js:
  LoadingScreen (judul game di atas, teks sambutan per huruf, tombol MULAI)
    → animasi whirlpool (#whirlpool + canvas)
    → main menu

game.js + player/world:
  MULAI → intro dialog (bingung di mana → sadar praaksara) di GameCore / dialog

dialog.js:
  StoryDialogs.intro       — baris demi baris dengan typewriter internal
  EndingDialogs.true/good/bad/secret — diputar di ending.js


RINGKASAN SISTEM YANG DIHUBUNGKAN SKRIP
----------------------------------------
- 10 soal utama: quiz.js (array 10 butir, acak, tidak mengulang indeks)
- Fakta setelah soal: Typewriter di factTxt (prefiks "Fakta: ")
- Ending: ending.js memeriksa misi + benar + secret + eksplorasi (flag di GS)
- Perpustakaan: library.js dari menu utama
- Crafting: crafting.js + tombol / hotkey
- Dimensi: dimension.js + interaksi E pada portal di world/npc


CATATAN MOBILE
--------------
viewport + kontrol sentuh ada di index.html (#controls, joystick).
